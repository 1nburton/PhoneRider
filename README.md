# PhoneRider

PhoneRider is an Expo React Native project based on a Line Rider style outline.

## Run locally

1. Install dependencies:
   npm install
2. Start Expo:
   npm run start

## Controls

- One finger: draw or erase track
- Two fingers: pan and pinch zoom
- Play/Stop controls in toolbar
- Demo tracks: tap DEMOS to load curated prebuilt runs
- Save slots: tap SAVES to store and reload local tracks
- Sound effects: "weeeee" on play, boost/brake/jump cues, and explosion on crash
